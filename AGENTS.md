# Agent instructions

This file tells AI coding agents (Codex, Claude, etc.) how to work in this repo.

## Commit messages

Follow **Conventional Commits** format:

    <type>: <imperative summary>

### Examples

- `exercise: add Tip Calculator`
- `project: add Malware Race`
- `docs: add README`
- `fix: correct output formatting in Tip Calculator`
- `chore: update .gitignore`

### Rules

- Use imperative mood -- "add" not "added", "fix" not "fixes"
- Keep the subject line under 72 characters
- Lowercase everything, including names and exercise titles
- One logical change per commit

### Type reference

- `exercise` - new exercise file from the course
- `docs` - README or documentation changes
- `fix` - bug fix or correction to existing code
- `notes` - personal notes or reflections
- `chore` - config, tooling, maintenance
- `refactor` - restructuring code without changing behaviour
